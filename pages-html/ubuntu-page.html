Ubuntu

Ubuntu merupakan salah satu distribusi Linux yang berbasis Debian dan didistribusikan sebagai perangkat lunak bebas atau open source. Ubuntu ditawarkan dalam tiga edisi resmi: Ubuntu Desktop untuk komputer pribadi, Ubuntu Server untuk server dan komputasi awan, dan Ubuntu Core untuk "IoT", perangkat kecil dan robot. Versi baru Ubuntu dengan dukungan standar dirilis tujuh bulan sekali, sementara untuk versi dukungan jangka-panjang (LTS) dirilis dua tahun sekali.

    Kelebihan

  • 1. Merupakan salah satu distribusi linux yang paling populer di dunia, dan juga memiliki banyak komunitas di seluruh dunia.
  • 2. Terdapat lebih dari 55 bahasa, termasuk bahasa Indonesia. Sehingga memudahkan anda dalam menggunakan Ubuntu, jika anda tak mengerti bahasa Inggris.
  • 3. Memiliki driver yang relatif lengkap.

    Kekurangan

  • 1. Harus Mengenali Terlebih Dahulu dan Adaptasi Jika Pindahan
  • 2. Ubuntu dektop lebih berat daripada server
  • 3. Kurang cocok untuk gaming berat
